# Smart Canteen Ordering System (Testing + Web Development)

## 📌 Project Overview
This project is a complete Smart Canteen Ordering System developed using PHP, HTML, CSS, and JavaScript, along with manual testing of all major functionalities.

---

## 🎯 Objective
- Develop a functional canteen ordering system
- Perform manual testing on all modules
- Identify and document bugs
- Validate user workflows and system behavior

---

## 💻 Web Development
The system includes:
- User authentication (Signup/Login/Logout)
- Shop listing and food items display
- Add to cart functionality
- Cart management
- Payment simulation
- Profile and password update

---

## 🧪 Testing Scope
Manual testing was performed on:
- Registration & Login validation
- Add to Cart & Cart behavior
- Payment functionality
- Profile & Password updates
- Navigation and UI behavior

---

## 🧾 Test Artifacts
- Test Scenarios
- Test Cases
- Bug Reports

📁 Available in: `Testing/Smart_Canteen_Testing_Project.xlsx`

---

## 🐞 Key Bugs Identified
- Invalid email accepted without domain
- Password accepts only spaces
- Transaction ID accepts alphabets/special characters
- Incorrect error message during shop switching
- Spaces allowed in username/login fields
